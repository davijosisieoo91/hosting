### What Is NOT Included in WP Engine Plans?

WP Engine covers managed hosting essentials, but some advanced or external services are not part of standard plans.

- Email hosting is not included, you need third-party providers like Google Workspace or Zoho Mail
- EV (Extended Validation) SSL certificates are not included, only free Let’s Encrypt SSL is provided by default
- WordPress multisite support is not available on lower tiers and requires a Growth plan or higher
- GeoIP-based content targeting features are only available starting from the Growth plan and above
- Smart Plugin Manager for automated plugin updates across sites is a paid add-on for most Essential plans
